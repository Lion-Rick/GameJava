# GameTouch2dFixel
 2dGameFixel
このゲームはAndroid Studioでjava言語を使って開発しました。

使い方：
    Android　Studioで実行してVirtual Devicesで表示します。
 

ゲームの概要:

    画面タッチでキャラクターを動かす。
    キャラクターから火を発射し、バットを破壊します。
    破壊したバットの数でスコアがアップします。
    キャラクターがバットに触れるとゲームオーバーになります。

ゲームの特徴:

    Androidの携帯で気軽にプレイします。
    タッチだけを使う簡単なインタフェースができます。
